Thực hiện các yêu cầu cho bài toán quản lý sinh viên. Yêu cầu làm trên ngôn ngữ lập trình Java (Console) , không yêu cầu code giao diện. Cho đối tượng sinh viên có các trường sau: Name (String) Age (int) Email (String) Phone (String) Code(String) Gender (int) - Giới tính Grade(float) - điểm Các hàm cần thực hiện Nhập/ Chèn mới thông tin sinh viên In danh sách sinh viên ra màn hình Xóa sinh viên theo mã code Sắp xếp sinh viên theo thứ tự điểm giảm dần Tìm kiếm sinh viên theo mã sinh viên hoặc tên sinh viên ( Có nghĩa là nhập tên hoặc mã sinh viên vào đều có thể tìm được) Tìm kiếm sinh viên có điểm >= x với x là giá trị nhập vào từ bàn phím

Dưới đây là mã nguồn Java cho bài toán quản lý sinh viên:

java
import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;
import java.util.Scanner;

class Student {
    private String name;
    private int age;
    private String email;
    private String phone;
    private String code;
    private int gender;
    private float grade;

    public Student(String name, int age, String email, String phone, String code, int gender, float grade) {
        this.name = name;
        this.age = age;
        this.email = email;
        this.phone = phone;
        this.code = code;
        this.gender = gender;
        this.grade = grade;
    }

    public String getName() {
        return name;
    }

    public String getCode() {
        return code;
    }

    public float getGrade() {
        return grade;
    }

    @Override
    public String toString() {
        return "Student{" +
                "name='" + name + '\'' +
                ", age=" + age +
                ", email='" + email + '\'' +
                ", phone='" + phone + '\'' +
                ", code='" + code + '\'' +
                ", gender=" + gender +
                ", grade=" + grade +
                '}';
    }
}

public class StudentManagement {
    private static ArrayList<Student> students = new ArrayList<>();
    private static Scanner scanner = new Scanner(System.in);

    public static void main(String[] args) {
        while (true) {
            System.out.println("1. Nhập thông tin sinh viên");
            System.out.println("2. In danh sách sinh viên");
            System.out.println("3. Xóa sinh viên theo mã code");
            System.out.println("4. Sắp xếp sinh viên theo điểm giảm dần");
            System.out.println("5. Tìm kiếm sinh viên theo mã hoặc tên");
            System.out.println("6. Tìm kiếm sinh viên có điểm >= x");
            System.out.println("7. Thoát");
            System.out.print("Chọn một tùy chọn: ");
            int choice = scanner.nextInt();
            scanner.nextLine(); // consume newline

            switch (choice) {
                case 1:
                    insertStudent();
                    break;
                case 2:
                    printStudents();
                    break;
                case 3:
                    deleteStudentByCode();
                    break;
                case 4:
                    sortStudentsByGrade();
                    break;
                case 5:
                    searchStudentByNameOrCode();
                    break;
                case 6:
                    searchStudentByGrade();
                    break;
                case 7:
                    System.exit(0);
                default:
                    System.out.println("Tùy chọn không hợp lệ!");
            }
        }
    }

    private static void insertStudent() {
        System.out.print("Nhập tên sinh viên: ");
        String name = scanner.nextLine();
        System.out.print("Nhập tuổi: ");
        int age = scanner.nextInt();
        scanner.nextLine(); // consume newline
        System.out.print("Nhập email: ");
        String email = scanner.nextLine();
        System.out.print("Nhập số điện thoại: ");
        String phone = scanner.nextLine();
        System.out.print("Nhập mã code: ");
        String code = scanner.nextLine();
        System.out.print("Nhập giới tính (0 - Nữ, 1 - Nam): ");
        int gender = scanner.nextInt();
        System.out.print("Nhập điểm: ");
        float grade = scanner.nextFloat();
        scanner.nextLine(); // consume newline

        students.add(new Student(name, age, email, phone, code, gender, grade));
    }

    private static void printStudents() {
        for (Student student : students) {
            System.out.println(student);
        }
    }

    private static void deleteStudentByCode() {
        System.out.print("Nhập mã code sinh viên cần xóa: ");
        String code = scanner.nextLine();

        students.removeIf(student -> student.getCode().equals(code));
    }

    private static void sortStudentsByGrade() {
        Collections.sort(students, (s1, s2) -> Float.compare(s2.getGrade(), s1.getGrade()));
        System.out.println("Danh sách sinh viên sau khi sắp xếp:");
        printStudents();
    }

    private static void searchStudentByNameOrCode() {
        System.out.print("Nhập tên hoặc mã code sinh viên cần tìm: ");
        String searchQuery = scanner.nextLine().toLowerCase();

        for (Student student : students) {
            if (student.getName().toLowerCase().contains(searchQuery) || student.getCode().toLowerCase().contains(searchQuery)) {
                System.out.println(student);
            }
        }
    }

    private static void searchStudentByGrade() {
        System.out.print("Nhập điểm tối thiểu: ");
        float minGrade = scanner.nextFloat();
        scanner.nextLine(); // consume newline

        for (Student student : students) {
            if (student.getGrade() >= minGrade) {
                System.out.println(student);
            }
        }
    }
}

Đây là chương trình quản lý sinh viên trên console với các chức năng nhập, in danh sách, xóa, sắp xếp và tìm kiếm sinh viên theo yêu cầu. Bạn có thể chạy chương trình này trong môi trường phát triển Java để kiểm tra và sử dụng.

  1. Mở IntelliJ IDEA.
  2. Chọn New Project.
  3. Chọn Java từ các loại project.

  1. Trong cửa sổ Project, nhấp chuột phải vào thư mục src.

  3. Đặt tên cho class là StudentManagement.

Bước 1: Lưu file trong IntelliJ IDEA

  1. Mở IntelliJ IDEA và đảm bảo rằng tất cả các file bạn đã chỉnh sửa đều được lưu. Bạn có thể nhấn Ctrl + S hoặc Cmd + S (trên macOS) để lưu file.
